			Написать телефонную книгу.
phone-book.js – здесь будет ваш код. В нём 2 метода, которые вы должны реализовать:
  add(name, phone, email) – добавляет запись в телефонную книгу, если какое-либо из полей не валидно, то запись не добавляется;
  remove(query) – удаляет запись из телефонной книги по запросу;
  search(string) - поиск по имени контакта
  reset() - сбросить поиск

invalid:
12korry

UPD. E-mail (от kvf77):
^[-\w.]+@([A-z0-9][-A-z0-9]+\.)+[A-z]{2,4}$

Регулярное выражение для валидации номера телефона:
^((8|\+7)[\- ]?)?(\(?\d{3}\)?[\- ]?)?[\d\- ]{7,10}$
Ориентировано на российские мобильные + 
городские с кодом из 3 цифр (например, Москва).
